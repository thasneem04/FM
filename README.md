# Fiction Master Website

A premium, cinematic, 3D-animated digital marketing agency website built with React + Vite and Framer Motion.

## Features

- **Cinematic Brand Entry**: Dynamic text reveal ("FICTION MASTER"), slogan animation, and 3D logo reveal.
- **Immersive Hero Section**: 3D floating background elements and bold typography.
- **Animated Services Grid**: Glassmorphic cards with 3D tilt effects on hover.
- **Responsive Navigation**: Glassmorphism navbar with mobile menu.
- **Dark Mode Aesthetic**: Luxury Red (#C4161C) and Pure White (#FFFFFF) on a deep cinematic background.
- **Performance Optimized**: Built with Vite for lightning-fast HMR and production builds.

## Getting Started

1.  **Install Dependencies** (already done):
    ```bash
    npm install
    ```

2.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization

-   **Colors**: Edit `src/styles/index.css` to modify CSS variables.
-   **Content**: Update components in `src/components/` to change text and images.
-   **Animations**: Adjusted in individual component files using Framer Motion props.

## Tech Stack

-   React 18
-   Vite
-   Framer Motion (Animations)
-   Lucide React (Icons)
